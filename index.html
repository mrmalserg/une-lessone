  • git clone path-to-repository (путь к репозиторию)
  • git status (статус локального репозитория)
  • git add index.html (это пример добавления нового файла под контроль GIT)
  • git commit -am "added new file index.html" (добавил в файл и закрепил)
  • git push - отправить все закрепленные (commit) изменения в репозиторий
  • git pull - забрать свежие обновления
